Busybox
=======


.. glossary::

    Description
        Tiny versions of common UNIX utilities built into a single binary

    Rationale
        It serves as a stand-in for bzip2, coreutils, diffutils, findutils, gawk, grep, gzip, sed, tar and util-linux
