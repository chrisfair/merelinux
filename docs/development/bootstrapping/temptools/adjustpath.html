11. Adjust PATHÂ¶

At this point, /mere should contain everything that is needed to successfully build the rest of the temporary tools. As a further protection from utilities in the host system being discovered and referenced in generated scripts, adjust the PATH environment variable to only contain /mere/bin.

The following script 1 outlines the commands required to complete this step:

#!/mere/bin/bash -e
sed -i '/^PATH=/s@=.*@=/mere/bin@' /mere/bin/clean_env
source /mere/bin/clean_env

# Verify that the current PATH is correct
test "/mere/bin" = "$(printf '%s' "$PATH")"
